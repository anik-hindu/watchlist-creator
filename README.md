# Watchlist Creator

Welcome! This is a simple web application that lets you create and manage a personal watchlist. It was built as a solo project after I completed the asynchronous JavaScript module in the [Scrimba Frontend Career Path](https://scrimba.com/).

### Live Preview: https://anik-hindu.github.io/watchlist-creator/

## About

I’m a beginner learning web development. This project helped me practice asynchronous JavaScript, DOM manipulation, and basic web app design. Everything here was coded by me as an individual learning exercise.

---

## Features

- **Add Items:** Add movies, series, or anything you plan to watch.
- **Remove Items:** Delete entries from your watchlist.
- **Responsive Design:** Works on desktops and mobile devices.
- **Client-side Storage:** Your data is kept in the browser (local storage).
